Please upload your final headshot image into this directory and name it `headshot.jpg`.

Guidance:
- Preferred dimensions: at least 800x800 px (square) for best results.
- Filename must be `headshot.jpg` to be used automatically by the homepage.
- After uploading, run `git add images/headshot.jpg && git commit -m "Add headshot" && git push`.

If you prefer a different filename, update `author.avatar` in `_config.yml` accordingly.
